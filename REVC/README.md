## REVC
[Complementing a Strand of DNA](https://rosalind.info/problems/revc/), is the third problem in [string algorithms](https://rosalind.info/problems/topics/string-algorithms/).

## Problem Statement
- Given: A DNA string _s_ of length at most 1000 bp.
- Return: The reverse complement _sc_ of _s_.

## How i solved
- Using **tr** to reverse the nucleotides as complementary.
- Then used **rev** command to reverse the output.
