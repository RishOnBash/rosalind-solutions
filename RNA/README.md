## RNA
[Transcribing DNA into RNA](https://rosalind.info/problems/rna/), is the second problem in [string algorithms](https://rosalind.info/problems/topics/string-algorithms/).

## Problem Statement
Given: A DNA string _t_ having length at most 1000 nt.
Return: The transcribed RNA string of _t_.

## How i solved
- Using **tr** to replace 'T' with 'U'.
